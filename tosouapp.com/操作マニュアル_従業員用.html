🔑 ログイン方法
推奨ブラウザ
  • Google Chrome（最新版）
  • Microsoft Edge（最新版）
  • Safari（最新版）
ログイン手順
  1. ログイン画面を開く
    https://tosouapp.com/ui/login
  2. ログインID を入力
    会社から配布されたID
  3. パスワード を入力
    会社から配布されたパスワード
  4. 「ログイン」ボタンを押す
  5. メニューから必要な画面を選択
⚠️ 注意: Edgeでログインできない、表示不具合が出る場合は、Google Chromeをご利用ください。
⏱️ 簡易打刻方法
毎日の基本運用（推奨）
  1. 出勤時に「開始打刻」を押す
    簡易登録画面で「開始打刻」ボタンをクリック
  2. 退勤時に「終了打刻」を押す
    「終了打刻」ボタンをクリック
  3. 当日または月内に月次勤怠画面で確認・補足入力する
  4. 必要に応じて現場・作業内容・理由などを入力して保存
「開始打刻」について
  • 出勤時に1回だけ押してください
  • 「開始打刻済み」と表示されたら、再度押さないでください
  • 時刻修正が必要な場合は、月次勤怠画面で修正してください
「終了打刻」について
  • 退勤時に押してください
  • 同じボタンを連続で何度も押さないでください
  • 時刻修正が必要な場合は、月次勤怠画面で修正してください
📋 月次勤怠入力
入力できる主な項目
  • 勤務区分 - 出勤、休日出勤、半休、休日、有給休暇など
  • 現場 - 担当している現場名
  • 作業内容 - 実施した業務内容
  • 開始時間 / 終了時間 - 勤務時間の修正
  • 休憩時間 - 休憩の時間設定
  • 理由 - 必要な場合に入力
保存方法
  • 行を変更すると自動保存される場合があります
  • 画面上部/下部の「保存」ボタンでも保存できます
  • 保存後に画面を再読込し、内容が反映されていることを確認してください
重要: 「出勤」や「休日出勤」を選択した場合、必要に応じて時刻も確認してください。
🛠️ トラブル対処
❌ 保存したのに戻る / 変わらない
対応方法:
  1. Ctrl + F5 で強制再読込
  2. 一度ログアウトして再ログイン
  3. Google Chromeで再度確認
❌ 打刻ボタンが押せない
対応方法:
  • すでに「開始打刻済み」の可能性があります
  • 画面を再読込（Ctrl + F5）してください
  • 時刻修正が必要な場合は、月次勤怠画面で行ってください
❌ 画面が遅い / 反映が遅い
対応方法:
  1. 5～10秒待機してから再確認
  2. 通信環境（Wi-Fi / 4G）を切り替え
  3. 不要なブラウザタブを閉じる
  4. 画面を再読込（Ctrl + F5）
💡 解決しない場合は？
直属管理者またはシステム担当へ以下の情報を含めて報告してください：
  • エラー画面のスクリーンショット
  • 発生日時と操作内容
  • 利用端末とブラウザ（Chrome / Edge等）
📱 スマートフォン利用
✅ スマートフォンで利用可能な機能
  • ログイン
  • 簡易打刻（開始・終了）
  • 月次勤怠入力の確認・修正
⚠️ 注意事項
  • 詳細な設定・管理操作はPC利用を推奨します
  • タテ向き（ポートレート）でのご利用をお勧めします
  • 通信環境が不安定な場合は、Wi-Fi接続をお勧めします
⚙️ 利用ルール
✅ 正しい操作方法
  • 打刻時に十分な間隔をあけてからボタンを押してください
  • 同じボタンを連続で何度も押さないでください
  • 定期的にパスワードを変更してください
  • 異常時は画面を撮影し、すぐに管理者へ報告してください
❌ 禁止事項
  • アカウントの共有 - 絶対に他人と共有しないでください
  • 他人のID・パスワード利用 - 自分のアカウントのみ使用してください
  • エラー時の連打操作 - エラー時は画面を再読込し、管理者に相談してください
  • 不正な打刻 - 実際の勤務と異なる入力は行わないでください
ご不明な点がございましたら、直属管理者またはシステム担当へお気軽にお問い合わせください。
