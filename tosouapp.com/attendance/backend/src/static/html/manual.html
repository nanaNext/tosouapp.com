🔑 ログイン方法
推奨ブラウザ
  • Google Chrome（最新版）
  • Microsoft Edge（最新版）
  • Safari（最新版）
ログイン手順
  1. ログイン画面を開く
    https://tosouapp.com/ui/login
  2. ログインID を入力
    会社から配布されたID
  3. パスワード を入力
    会社から配布されたパスワード
  4. 「ログイン」ボタンを押す
  5. メニューから必要な画面を選択
⚠️ 注意: Edgeでログインできない、表示不具合が出る場合は、Google Chromeをご利用ください。
⏱️ 簡易打刻方法
毎日の基本運用（推奨）
  1. 出勤時に「開始打刻」を押す
    簡易登録画面で「開始打刻」ボタンをクリック
  2. 退勤時に「終了打刻」を押す
    「終了打刻」ボタンをクリック
  3. 当日または月内に月次勤怠画面で確認・補足入力する
  4. 必要に応じて現場・作業内容・理由などを入力して保存
「開始打刻」について
  • 出勤時に1回だけ押してください
  • 「開始打刻済み」と表示されたら、再度押さないでください
  • 時刻修正が必要な場合は、月次勤怠画面で修正してください
「終了打刻」について
  • 退勤時に押してください
  • 同じボタンを連続で何度も押さないでください
  • 時刻修正が必要な場合は、月次勤怠画面で修正してください
📋 月次勤怠入力
入力できる主な項目
  • 勤務区分 - 出勤、休日出勤、半休、休日、有給休暇など
  • 現場 - 担当している現場名
  • 作業内容 - 実施した業務内容
  • 開始時間 / 終了時間 - 勤務時間の修正
  • 休憩時間 - 休憩の時間設定
  • 理由 - 必要な場合に入力
保存方法
  • 行を変更すると自動保存される場合があります
  • 画面上部/下部の「保存」ボタンでも保存できます
  • 保存後に画面を再読込し、内容が反映されていることを確認してください
重要: 「出勤」や「休日出勤」を選択した場合、必要に応じて時刻も確認してください。
💴 給与管理（管理者向け）
利用画面
管理ポータルの 給与管理 から操作します。対象社員・対象年月を選び、 保存 → プレビュー → PDF作成 → PDFダウンロード → 社員へ送信（公開） の順で進めるのが基本です。
基本操作の流れ
  1. 社員選択 と 対象年月 を指定します。
  2. 基本給（月給） を入力します。
  3. 必要に応じて 自動計算（保険・税） を選択します。
  4. 勤怠情報、支給、控除、その他、支払方法を確認し、必要な箇所だけ手入力で調整します。
  5. 保存 を押して月次給与データを保存します。
  6. プレビュー で内容確認後、問題なければ PDF作成 を実行します。
  7. 作成後に PDFダウンロード で帳票確認し、問題なければ 社員へ送信（公開） を押します。
画面ごとの意味
  • 総計（リアルタイム） - 総支給額、総控除額、手取り、支払方法合計をその場で確認します。
  • 勤怠情報 - 出勤日数、欠勤日数、就業時間、法外時間外、週40超時間、月60超時間、深夜勤務時間を確認・補正します。
  • 支給 - 基本給、交通手当、時間外手当、深夜勤手当、欠勤控除、追加支給などを確認します。
  • 控除 - 社会保険、税金、家賃控除、追加控除を確認します。
  • 支払方法 - 振込、現金、現物の内訳を設定します。合計は手取り額と一致させてください。
給与計算の考え方
1. 勤怠の集計
  • 出勤日数は日次勤怠と打刻実績から自動集計されます。
  • 有給休暇・欠勤・無給休暇は、日次区分と申請データの両方を参照して反映します。
  • 就業時間、時間外、深夜、週40超、月60超は勤怠実績から自動計算されます。
  • 画面で手入力した値がある場合は、手入力値を優先します。
2. 支給額の計算
  • 基礎給 は基本給（月給）です。
  • 就業手当 は交通手当欄の値または社員設定値を使います。
  • 時間外手当 / 深夜勤手当 / 所休出手当 / 週40超手当 / 月60超手当 / 法休出手当 は、勤怠時間と会社設定の割増率から自動計算されます。
  • 欠勤控除 は、欠勤日数 + 無給休暇があり、手入力の欠勤控除がない場合に自動計算されます。
  • 追加支給、差額計算、年末調整還付などは必要に応じて加算されます。
3. 控除額と手取り
  • 自動計算（保険・税）= する にした場合、健康保険、介護保険、厚生年金、雇用保険、所得税が自動計算されます。
  • 住民税、家賃控除、追加控除、年末調整徴収、追加診療費などは必要に応じて調整します。
  • 総支給額 - 総控除額 = 差引支払額（手取り） です。
  • 振込 + 現金 + 現物 = 支払方法合計 となり、手取り額と一致している必要があります。
運用上の注意
  • まず勤怠が正しいか確認してから給与入力を行ってください。
  • 基本給や保険税の設定を変えた後は、必ず プレビュー で再確認してください。
  • PDF作成後は、そのPDFを開いて文字化け・金額・社員名・年月を確認してください。
  • 社員へ送信（公開） を押すと、社員側マイページから閲覧可能になります。
チェックリスト
  • 対象社員・対象年月が正しい
  • 総支給額、総控除額、手取りが想定どおり
  • 振込・現金・現物の合計が手取りと一致
  • プレビュー確認済み
  • PDF作成・PDFダウンロード確認済み
  • 問題なければ社員へ送信（公開）
🛠️ トラブル対処
❌ 保存したのに戻る / 変わらない
対応方法:
  1. Ctrl + F5 で強制再読込
  2. 一度ログアウトして再ログイン
  3. Google Chromeで再度確認
❌ 打刻ボタンが押せない
対応方法:
  • すでに「開始打刻済み」の可能性があります
  • 画面を再読込（Ctrl + F5）してください
  • 時刻修正が必要な場合は、月次勤怠画面で行ってください
❌ 画面が遅い / 反映が遅い
対応方法:
  1. 5～10秒待機してから再確認
  2. 通信環境（Wi-Fi / 4G）を切り替え
  3. 不要なブラウザタブを閉じる
  4. 画面を再読込（Ctrl + F5）
💡 解決しない場合は？
直属管理者またはシステム担当へ以下の情報を含めて報告してください：
  • エラー画面のスクリーンショット
  • 発生日時と操作内容
  • 利用端末とブラウザ（Chrome / Edge等）

✅ スマートフォンで利用可能な機能

  • 簡易打刻（開始・終了）

⚠️ 注意事項
  • 詳細な設定・管理操作はPC利用を推奨します
  • タテ向き（ポートレート）でのご利用をお勧めします
  • 通信環境が不安定な場合は、Wi-Fi接続をお勧めします
⚙️ 利用ルール
✅ 正しい操作方法
  • 打刻時に十分な間隔をあけてからボタンを押してください
  • 同じボタンを連続で何度も押さないでください
  • 定期的にパスワードを変更してください
  • 異常時は画面を撮影し、すぐに管理者へ報告してください
❌ 禁止事項
  • アカウントの共有 - 絶対に他人と共有しないでください
  • 他人のID・パスワード利用 - 自分のアカウントのみ使用してください
  • エラー時の連打操作 - エラー時は画面を再読込し、管理者に相談してください
  • 不正な打刻 - 実際の勤務と異なる入力は行わないでください
ご不明な点がございましたら、直属管理者またはシステム担当へお気軽にお問い合わせください。
