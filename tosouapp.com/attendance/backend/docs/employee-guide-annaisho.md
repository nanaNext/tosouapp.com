# 案内書（社員向け）

## 1) 目的
本書は、`tosouapp.com` で日々の勤怠打刻を行うための社員向け手順書です。

## 2) 公式利用リンク
- ログイン画面: `https://tosouapp.com/ui/login`
- 簡易打刻画面: `https://tosouapp.com/ui/attendance/simple`

## 3) 打刻前の準備
- Chrome / Edge / Safari の最新版を利用してください。
- インターネット接続が安定していることを確認してください。
- 会社の正しいアカウントでログインしてください。

## 4) 標準打刻手順
1. システムにログインします。
2. `簡易登録画面` を開きます。
3. 必要に応じて `勤務区分` と `勤務形態` を選択します。
4. 勤務開始時に `開始打刻` を押します。
5. 勤務終了時に `終了打刻` を押します。
6. `開始時間 / 終了時間 / 実務時間` を確認します。

## 5) よくある事象と一次対応
- `開始打刻` が押せない:
  - `勤務形態` が選択済みか確認する。
  - 画面を再読み込みする（`Ctrl + F5`）。
  - 一度ログアウトして再ログインする。
- 画面表示が遅い:
  - 5-10秒待つ。
  - 4G/Wi-Fi を切り替える。
  - 不要なブラウザタブを閉じる。
- 表示状態が最新でない:
  - `再読込` を押す。
  - 再ログインする。

## 6) 操作ルール
- 同じボタンを連続で何度も押さないでください。
- アカウントの共有は禁止です。
- 異常時は画面を撮影し、すぐ管理者へ報告してください。

## 7) サポート連絡
- 直属管理者またはシステム担当へ連絡してください。
- 報告時は以下を共有してください:
  - エラー画面のスクリーンショット
  - 発生時刻
  - 利用端末 / ブラウザ
